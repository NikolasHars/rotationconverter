# 3D Rotation Converter - Enhanced Features

## 🎮 Keyboard Controls
- **Arrow Keys / WASD**: Move camera forward/backward/left/right
- **Q / E**: Move camera up/down
- **R**: Reset camera to origin view

## 👁️ Frame Visibility Controls
- **Toggle Frames**: Hide/show all coordinate frame axes
- **Toggle Children**: Hide/show only child frame axes (keeps objects visible)

## 📐 Enhanced 3D Visualization
- **Larger Panel**: 3D visualization now takes up much more screen space
- **Sticky Panels**: Control panels stay visible when scrolling
- **Responsive Design**: Adapts to different screen sizes

## 🖱️ Mouse Controls (unchanged)
- **Left Click + Drag**: Rotate view
- **Right Click + Drag**: Pan view
- **Scroll Wheel**: Zoom in/out

## 💡 Usage Tips
- Use keyboard controls for quick camera navigation
- Mouse controls are best for precise adjustments
- Press R anytime to return to a good overview of the origin
- Frame visibility controls help reduce visual clutter when working with complex hierarchies
- The visualization panel is now much larger for better 3D work

## 🎯 Perfect for:
- Visualizing complex frame hierarchies
- Working with COLMAP point clouds
- Analyzing 3D transformations
- Educational demonstrations of coordinate systems
